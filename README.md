# Neon Vault Playable Ad

A lightweight, Vercel-friendly playable ad prototype for a mobile puzzle game.

## Concept

`Neon Vault` is a modern circuit-rotation puzzle designed to feel quick, tactile, and satisfying inside an ad placement. The player rotates glowing tiles to connect the energy source to the vault, gets immediate visual payoff, and is then presented with a strong install CTA.

## Why this setup

- No framework or build step required
- Static hosting friendly for Vercel
- Mobile-first layout with touch interactions
- Fast load time and simple repo structure

## Local preview

You can open `index.html` directly in a browser, or serve the folder with any static file server.

## Deploy to Vercel

This project is ready to deploy as a static site on Vercel with no additional configuration beyond importing the repository.
